Well this presentation is a slide Editor too !!!

Read on to see how you can edit and create new presentations

Slide Toolbar

  • Press Alt+T to show or Hide the toolbar
  • Toolbar is the starting point
  • You can drag the toolbar
  • The Toolbar consists of many features
  • Go ahead and explore them or just read on

Add or Delete Slide

  • Press "Add Slide" button to add a New slide
  • The newly added slide will have some dummy content
  • Pressing Delete button will remove the current slide

EDIT Slide

  • "Edit Slide" button will make the focused slide editable
  • Now, you can click anywhere on the slide and edit it.
  • After editting press the "Save Slide" button.
  • If you are an HTML fan then you are just in luck.
  • You can use the "Raw Edit" button.
  • Now you can write any HTML /CSS /JS code and it will be reflected in the slide.
  • The Slide Editor can be dragged and to close it press done

Drag and Drop Slides

  • It is very easy to shuffle your slide ordering too.
  • First press ESC key
  • Now click on "Drag and Drop" button
  • This will allow the slides to be dragged
  • Feel free to drag and drop any slide anywhere

Save the Presentation

  • Currently we are not saving the presentation on our servers, because of copyright and server space restrictions. But it can be easily done.
  • Instead we just give you the code.
  • Just hit the "Show me the Code" button
  • And get the code for your whole presentation
